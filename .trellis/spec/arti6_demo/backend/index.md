# ARTI6 Demo Backend Guidelines

## Scope

Applies to `arti6_linearvc_demo/`, which remains the ARTI-6 + LinearVC research
demo lane.

## Pre-Development Checklist

Read these before editing the ARTI6 demo:

- `.trellis/spec/repo-structure.md`
- `.trellis/spec/arti6-linearvc-acceptance.md`
- `.trellis/spec/backend/directory-structure.md`
- `.trellis/spec/backend/quality-guidelines.md`
- `.trellis/spec/backend/logging-guidelines.md`

## Local Rules

- Keep this lane focused on ARTI-6 / LinearVC reproduction, smoke tests, reports,
  and small reviewed demo artifacts.
- Do not mix VTuber data curation utilities into this lane; use
  `vtuber_pipeline/` for that work.
