# prd.md: Visualize Sound Processing Report

## Goal
Create an HTML visualization of the `sound_processing_report.md` to teach the user complex signal processing concepts (BSS, ICA, IVA, ILRMA, MM Algorithm) using analogies and visual structure.

## Requirements
* Interactive Slide Deck using Reveal.js.
* Use analogies (e.g., Cocktail Party Problem, painting vs mixing paint).
* Simplify the math so the intuition is clear before the equations.

## Acceptance Criteria
* [x] HTML file generated at `reports/sound_processing_visualized.html`.
