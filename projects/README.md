# Project Lanes

This directory is an index for the workspace-level split. It does not store
large data or generated model artifacts.

## Lanes

- `vtuber_voice/`: Current VTuber speech/singing data curation work. See
  `vtuber_voice/progress.md` for current status.
- `arti6_linearvc/`: Original ARTI-6 + LinearVC demo work.
- `trellis_harness/`: Local Trellis and AI-platform integration work.

The executable code currently remains in the established top-level directories
(`vtuber_pipeline/`, `arti6_linearvc_demo/`, `.trellis/`, `.agents/`, `.agent/`)
so existing commands keep working.
