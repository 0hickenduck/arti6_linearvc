Linear Speaker Mapping (Articulation 6)

LinearVC projects standard acoustic parameters (like Mel-spectrograms) between speakers using simple linear translations or affine transformations. The core research hypothesis is that voice conversion can be achieved by shifting the acoustic manifold while preserving local structures representing articulation.

1

Mean-Standard Deviation Shift: Translates and scales the acoustic envelope globally. Extremely simple, yet preserves standard phonetic structures.

2

Diagonal Affine Transform: Independent scale and offset per feature dimension, providing a personalized adjustment of frequency bands.

3

Full Affine Transformation: Includes rotation and covariance mapping, aligning the feature manifolds across speakers globally.

Affine Speaker Transform
y_t = W * x_t + b

W = diagonal or full matrix
b = translation vector

Timbre Hybrids
z_t = α * y_t + (1 - α) * oracle_vocal

Timbre Pivoting Voice Conversion (Seed-VC)

Seed-VC is a non-linear diffusion/flow-matching voice conversion model. Using the newly curated dry VTuber dataset, we perform zero-shot and few-shot voice conversion. By inputting a source singing track along with a target speech/singing reference, the model pivots the singer's timbre to match the target speaker.

1

Speech-to-Singing timbre pivoting: Allows using standard chatting/talking voice references to guide highly dynamic singing voice conversions.

2

Cross-Lingual Adaptation: Translates vocal timbre from English speakers to Japanese targets perfectly while maintaining phonetic clarity.

3

Zero-shot Inference: Leverages deep semantic bottlenecks (like content representations) to map voices in a speaker-agnostic feature space.

Seed-VC Bottlenecks
Speech = Content + Pitch + Timbre

Content extractor = vits / whisper
Timbre reference = target speech

Pivot Logic
z_latent = Flow-Match(source_singing_content, target_speech_reference)
