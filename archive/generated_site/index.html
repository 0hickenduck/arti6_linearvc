ARTI-6 + LinearVC

Can a voice change live in six moving numbers?

This project tests whether speaker-dependent voice traits can be represented as explicit linear transformations of low-dimensional articulatory trajectories instead of hiding everything inside a black-box embedding.

speech → articulation → speech interpretable VC 6D vocal-tract state

What ARTI-6 gives us

ARTI-6 converts speech into six interpretable articulatory channels, then synthesizes audio back from those channels plus a speaker embedding. The upstream model reports a 0.87 inversion correlation and supports both inversion and synthesis.

The research question

Instead of asking only whether a target speaker embedding changes the output voice, we ask a sharper question:

Does transforming the 6D articulatory trajectory itself add speaker information beyond the target embedding alone?

If yes, the system becomes easier to inspect, compare, and reason about than a purely latent conversion pipeline.

Eventual pipeline

1
Invert source and target speech into (T, 6) articulatory trajectories plus speaker embeddings.
2
Reconstruct each speaker from their own features to verify the base model.
3
Build the baseline: source_arti + target_spk_emb.
4
Fit linear transforms such as W · source_arti + b, then synthesize transformed variants.

The decisive comparison

Baseline

source_arti_feats + target_spk_emb

Proposed

transformed_source_arti + target_spk_emb

If the proposed path sounds more target-like than the baseline, the articulatory transform is doing real work rather than merely riding on the target embedding.

What we will save

  • original and reconstructed wav files
  • embedding-only and transformed VC audio
  • .npy feature tensors
  • trajectory plots and affine heatmaps

Current state

The repo is still in staged research mode: the safe local plan is written, the ARTI-6 code is vendored, and the next gates focus on environment checks, smoke tests, then tiny paired experiments.

Why this matters

A six-dimensional control surface is small enough for humans to inspect. If it preserves meaningful speaker variation, voice conversion becomes less like alchemy and more like engineering.
